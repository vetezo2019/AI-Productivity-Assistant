AI Workplace Productivity Assistant
📌 Project Overview

AI Workplace Productivity Assistant is a modern, responsive SaaS-style web application designed to help professionals automate everyday workplace tasks using Artificial Intelligence.
The platform centralizes multiple AI-powered productivity tools into a single, clean dashboard interface, enabling users to work smarter, faster, and more efficiently.

The application focuses on usability, responsible AI usage, and scalable architecture, making it suitable for real-world professional environments.

🚀 Features
1. Smart Email Generator
Generates professional emails based on user inputs such as purpose, tone, and audience
Editable AI-generated content
Option to regenerate or refine responses
2. Meeting Notes Summarizer
Converts meeting transcripts or notes into concise summaries
Highlights key points, decisions, and action items
Improves post-meeting productivity and clarity
3. AI Task Planner
Automatically creates task lists from project descriptions or goals
Suggests task prioritization and deadlines
Helps users stay organized and focused
4. AI Research Assistant
Summarizes research topics or provided text
Produces structured insights for easy understanding
Ideal for quick knowledge gathering and analysis
5. AI Chatbot Interface
Conversational assistant for general workplace support
Context-aware responses
Professional and productivity-focused interactions

⚙️ Setup Instructions
1. Clone the Repository
git clone https://github.com/your-username/ai-workplace-productivity-assistant.git
cd ai-workplace-productivity-assistant
2. Install Dependencies
npm install
3. Environment Variables

Create a .env file in the root directory and add:

AI_API_KEY=your_api_key_here

⚠️ Do not expose API keys publicly.

4. Run the Application
npm run dev

The application will be available at:

http://localhost:3000
